## Création du compte

Pour commencer à utiliser Première Copilot, vous devez d'abord créer un compte utilisateur.

<Steps>
  <Step title="Accédez au site">
    Rendez-vous sur le [site Première Copilot](https://premierecopilot.com).
  </Step>
  <Step title="Lancez la procédure">
    Cliquez sur le bouton **« Download for free »** en haut à droite de la page.
  </Step>
  <Step title="Renseignez vos informations">
    Remplissez le formulaire avec votre prénom, nom et les autres informations requises.
  </Step>
  <Step title="Inscrivez-vous">
    Cliquez sur **« Sign up »**.
  </Step>
  <Step title="Vérifiez votre e-mail">
    **Vérifiez votre e-mail :** ouvrez votre boîte de réception et validez votre compte en cliquant sur le lien de confirmation.
  </Step>
</Steps>

<Note>
  **Note :** à l'inscription, vous recevrez un code promo pour acheter le plugin « Podcast » à tarif réduit.
</Note>

## Accès au tableau de bord

Une fois vérifié, retournez sur le site et cliquez sur **« Sign in »** pour accéder à votre tableau de bord. Depuis celui-ci, vous pouvez :

<CardGroup cols={2}>
  <Card title="Télécharger l'application" icon="download">
    Téléchargez la dernière version de l'application.
  </Card>
  <Card title="Réserver une démo" icon="calendar">
    Planifiez une démo pour découvrir plus de fonctionnalités.
  </Card>
  <Card title="Rejoindre Discord" icon="discord">
    Rejoignez notre communauté sur Discord.
  </Card>
  <Card title="Code affilié" icon="ticket">
    Accédez à votre code promo pour vos parrainages.
  </Card>
</CardGroup>
